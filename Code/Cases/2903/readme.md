### 题目描述

给定一个字符串数组 arr，字符串 s 是将 arr 某一子序列字符串连接所得的字符串，如果 s 中的每一个字符都只出现过一次，那么它就是一个可行解。

请返回所有可行解 s 中最长长度。

### 输入描述

```
一个字符串数组 arr
1 <= arr.length <= 16
1 <= arr[i].length <= 26
arr[i] 中只含有小写英文字母
```
### 输出描述

```
返回所有可行解 s 中最长长度。
```

### 测试样例
#### 样例1: 输入-输出-解释
```
["un","iq","ue"]
```
```
4
```
```
所有可能的串联组合是 "","un","iq","ue","uniq" 和 "ique"，最大长度为 4。
```
