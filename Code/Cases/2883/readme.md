### 题目描述

给你一个 n 行 m 列的白色矩阵，这个白色矩阵中存在一个边长为奇数的黑色子矩阵，请你求出这个黑色子矩阵的中心位置的坐标。

### 输入描述

```
第一行输入行数 n，列数 m (1 <= n,m <= 115)
接下来输入矩阵，矩阵保证符合题目描述，即保证有解。'W'代表白色，'B'代表黑色
```

### 输出描述

```
输出两个整数x，y。表示黑色子矩阵的中心位置在第 x 行第 y 列
```

### 测试样例

#### 样例1: 输入-输出

```
5 6
WWBBBW
WWBBBW
WWBBBW
WWWWWW
WWWWWW
```

```
2 4
```

#### 样例2: 输入-输出

```
3 3
WWW
BWW
WWW
```

```
2 1
```
