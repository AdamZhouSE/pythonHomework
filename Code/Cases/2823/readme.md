### 题目描述

给你长度为 l 的整数数列 (1 ≤ b1 ≤ b2 ≤ ... ≤ bl ≤ *n*)，如果每个元素都可以整除下一个元素，那么我们称这个数列是好的。给你 n 和 k，请找到长度为 k 的好数列的个数吧，如果它特别大的话，请用 MOD=1000000007 取模。

### 输入描述

```
只有一行，为数列元素的最大值 n 和长度 k (1 ≤ n, k ≤ 2000)
```

### 输出描述

```
输出一个整数表示长度为 k 的好数列的个数，用 1000000007 取模
```

### 测试样例

#### 样例1: 输入-输出

```
3 2
```

```
5
```

#### 样例2: 输入-输出

```
6 4
```

```
39
```

#### 样例3: 输入-输出

```
2 1
```

```
2
```
