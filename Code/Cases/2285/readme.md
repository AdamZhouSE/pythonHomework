### 题目描述
每天的股票成本以大小N的数组A []给出。找到购买和出售股票的所有天数，以便在这两天之间您的利润最大化。
### 输入描述
第一行包含测试用例的数量T.每个测试用例的第一行包含表示天数的整数N，其后是N天的股票价格数组。
### 输出描述
对于每个测试用例，将一行全天输出有利润。如果没有利润，则打印``没有利润''。
### 输入范例
```
2
7
100 180 260 310 40 535 695
10
23 13 25 29 33 19 34 45 65 67
```
### 输出范例
```
(0 3) (4 6)
(1 4) (5 9)
```
