### 题目描述

给一个n(1<=n<=2500)个点m(1<=m<=6200)条边的无向图，求s到t的最短路。

### 输入描述

```
第一行四个由空格隔开的整数n、m、s、t。
之后的m行，每行三个正整数s_i（符号“_”表示下标）、t_i、w_i(1<=W_i<=10^9)，表示一条从s_i到t_i长度为w_i的边。

```
### 输出描述

```
一个整数表示从s到t的最短路长度。数据保证至少存在一条道路。
```
