
## 题目描述
对于给定的非负整数N，找到下一个最小的快乐数。如果一个数字在一系列步骤后均以1表示为1，则在每个步骤中，数字均用其数字的平方和代替，即如果我们以Happy Number开头并继续用数字的平方和代替，则得出1。
## 输入描述
输入的第一行是整数T，表示测试用例的数量。对于每个测试用例，有一行输入由整数N组成，必须找到其下一个最小的幸福数。0<=T<=200;0<=N<1000
## 输出描述
每个测试用例的唯一输出行包括给定数字N的下一个最小快乐数。如果整数N i本身就是一个快乐数，那么它的下一个最小快乐数也应作为输出打印。
## 样例输入
4

8

3

0

10
## 样例输出
10

7

1

13
