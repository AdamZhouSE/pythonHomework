### 题目描述
给定正整数数组。您的任务是在阵列中找到领导者。 注意：如果array的一个元素大于或等于其右侧的所有元素，则为leader。同样，最右边的元素始终是领导者。
### 输入描述
输入的第一行包含一个整数T，表示测试用例的数量。 T测试用例的描述如下。 每个测试用例的第一行包含一个表示数组大小的整数N。 第二行包含N个以空格分隔的整数A1，A2，...，AN，它们表示数组的元素。
### 输出描述
打印所有领导者
### 输入范例
```
3
6
16 17 4 3 5 2
5
1 2 3 4 0
5
7 4 5 7 3
```
### 输出范例
```
17 5 2
4 0
7 7 3
```
