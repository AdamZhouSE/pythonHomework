### 题目描述

Erwin最近对一种叫"thair"的东西巨感兴趣。。。

在含有n个整数的序列a1,a2......an中，

三个数被称作"thair"当且仅当i<j<k且ai<aj<ak

求一个序列中"thair"的个数。


### 输入描述

```
开始一个正整数n,

以后n个数a1~an。
```
### 输出描述

```
"thair"的个数
```

### 测试样例
#### 样例1:输入-输出-解释

```
4
2 1 3 4
```
```
2
```
```
无
```
#### 样例2: 输入-输出-解释
```
5
1 2 2 3 4
```
```
7
```
```
7个"thair"分别是
1 2 3 1 2 4 1 2 3 1 2 4 1 3 4 2 3 4 2 3 4
```
