### 题目描述
给定一个由N个正数和负数组成的数组a []。查找是否存在一个总和为0的子数组（大小至少为一个）。
### 输入描述
输入的第一行包含一个整数T，表示测试用例的数量。然后是T测试用例。每个测试用例都包含一个整数n，它表示数组的大小。下一行包含形成数组的N个空格分隔的整数。
### 输出描述
如果存在大小不小于1且总和等于0的子数组，则打印“Yes”（不带引号），否则打印“No”（不带引号）。
### 输入范例
```
2
5
4 2 -3 1 6
5
4 2 0 1 6
```
### 输出范例
```
Yes
Yes
```
