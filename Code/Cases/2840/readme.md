### 题目描述

罗玛的幸运数字是 4 和 7，因此任何由这两个数字组成的数都是他的幸运数。

罗玛现在得到了 n 个数，他想知道有多少个数中幸运数字的个数不超过 k 个。

例如 k = 2，[4,477,5] 中，[4,5] 两个数的幸运数字为 [1,0] 不超过两个，而 477 有三个。

### 输入描述

```
输入第一行为题目所述的 n 和 k (1 ≤ n, k ≤ 100)
第二行为罗玛得到的 n 个数 (1 ≤ ai ≤ 109)
```

### 输出描述

```
输出有多少个数的幸运数字不超过 k 个
```

### 测试样例

#### 样例1: 输入-输出

```
3 4
1 2 4
```

```
3
```

#### 样例2: 输入-输出

```
3 2
447 44 77
```

```
2
```
