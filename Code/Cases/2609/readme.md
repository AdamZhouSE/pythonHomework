### 题目描述
给定一个由N个整数组成的数组，编写一个程序以打印数组中第K个不同的元素。给定的数组可能包含重复项，并且输出应在所有唯一元素中打印第K个元素。如果K大于不同元素的数量，则打印-1。
### 输入描述
输入的第一行包含一个整数T，它表示测试用例的数量。每个测试用例的第一行包含两个空格分隔的整数N和K。每个测试用例的第二行包含N个空格分隔的整数。
### 输出描述
对于每个测试用例，如上所述，打印第K个不同的元素。
### 输入范例
```
3
6 2
1 2 1 3 4 2
6 3
1 2 50 10 20 2
4 2
2 2 2 2
```
### 输出范例
```
4
10
-1
```
