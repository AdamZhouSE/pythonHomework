### 题目描述

有 N 双袜子，第 i 双袜子的每只袜子编号为 i。现在知道从袋子中拿取袜子的顺序，袜子一只一只拿。如果桌子上有配对的袜子，就将这双袜子放进柜子里；如果没有，就将袜子放在桌子上。问桌子上最多存在几只袜子？

### 输入描述

```
第一行一个 N (1 <= N = 100000)
第二行 2*N 个整数，xi 表示第 i 次从袋子中取出的袜子编号为 xi(1<=xi<=N)
```

### 输出描述

```
输出桌子上最多存在的袜子数量
```

### 测试样例

#### 样例1: 输入-输出

```
1
1 1
```

```
1
```

#### 样例2: 输入-输出

```
3
2 1 1 3 2 3
```

```
2
```
