### 题目描述

L发明了一种与树有关的游戏（友情提醒：树是一个没有环的连通图）：他从树中删除任意数量（可以为0）的边，计算删除后所有连通块大小的乘积，L将得到这么多的分数。你的任务就是对于一颗给定的树，求出L能得到的最大分数。


### 输入描述

```
第一行一个整数n，表示树的节点个数。

接下来n-1行，每行两个整数a[i],bi，表示a[i]与b[i]之间连边。

保证输入的图是一棵树。
```
### 输出描述

```
输出一个整数，表示L能得到的最大分数。
```

### 测试样例
#### 样例1:输入-输出-解释

```
5
1 2
2 3
3 4
4 5
```
```
6
```
```
无
```
