### 题目描述

在二维平面上，我们将石头放置在一些整数坐标点上。每个坐标点上最多只能有一块石头。

现在，move 操作将会移除与网格上的某一块石头共享一列或一行的一块石头。

我们最多能执行多少次 move 操作？

### 输入描述

```
一个二维平面。比如：stones = [[0,0],[0,1],[1,0],[1,2],[2,1],[2,2]]
1 <= stones.length <= 1000
0 <= stones[i][j] < 10000
```
### 输出描述

```
执行 move 操作的次数
```

### 测试样例
#### 样例1: 输入-输出-解释
```
[[0,0],[0,1],[1,0],[1,2],[2,1],[2,2]]
```
```
5
```
