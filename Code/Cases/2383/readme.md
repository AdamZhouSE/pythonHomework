### 题目描述

给出N个点的树和K，问能否把树划分成N/K个连通块，且每个连通块的点数都是K。


### 输入描述

```
第1行，1个整数T，表示数据组数。接下来T组数据，对于每组数据：
第1行，2个整数N,K。
接下来N−1行，每行2个整数A_i,B_i，表示边A_i,B_i。点用1,2,...,N编号。
```
### 输出描述

```
对于每组数据，输出YES或NO。
```

### 测试样例
#### 样例1:输入-输出-解释

```
2
4 2
1 2
2 3
3 4
4 2
1 2
1 3
1 4
```
```
YES
NO
```
```
无
```
