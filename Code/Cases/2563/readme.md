### 题目描述

对于给定的整数 n, 如果n的k（k>=2）进制数的所有数位全为1，则称 k（k>=2）是 n 的一个好进制。

以字符串的形式给出 n, 以字符串的形式返回 n 的最小好进制。

### 输入描述

```
以字符串的形式给出 n。n的取值范围是 [3, 10^18]。输入总是有效且没有前导 0。
```
### 输出描述

```
以字符串的形式返回 n 的最小好进制。
```

### 测试样例
#### 样例1: 输入-输出-解释
```
"13"
```
```
3
```
```
13 的 3 进制是 111。
```
#### 样例2: 输入-输出-解释
```
"1000000000000000000"
```
```
"999999999999999999"
```
```
1000000000000000000 的 999999999999999999 进制是 11。
```
