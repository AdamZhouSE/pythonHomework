### 题目描述

波利卡普有 n 个硬币，每个硬币的面值为 ai，他想将这些硬币装到多个口袋里，使得每个口袋中硬币的面值都不同，比如 [1, 2, 4, 3, 3, 2] 六个硬币，至少需要装入到两个口袋中，分别为 [1, 2, 3] 和 [2, 3, 4]。

波利卡普想知道至少需要多少个口袋才能装下这些硬币，请你帮助他。

### 输入描述

```
输入第一行为硬币数 n (1≤n≤100) 
第二行为这 n 个硬币的面值 (1≤ai≤100)
```

### 输出描述

```
输出一个整数表示至少需要的口袋数
```

### 测试样例

#### 样例1: 输入-输出

```
6
1 2 4 3 3 2
```

```
2
```

#### 样例2: 输入-输出

```
1
100
```

```
1
```
