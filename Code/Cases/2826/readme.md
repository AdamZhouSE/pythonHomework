### 题目描述

小图有 *n* 枚奖章。他想给他的 *n* 个士兵每人一枚奖章。 每枚奖章都有对应的卓越指数，表示获得者的卓越程度。 

但是目前的问题是有些奖章的卓越指数是一样的，小图希望每个士兵都应该获得不同卓越指数的奖章，卓越指数的精确值并不重要，只需要各不相同即可。

若一个硬币能把卓越指数提升一点，请帮他计算一下，他最少需要花多少个硬币才能让每枚奖章获得不一样的卓越指数。

### 输入描述

```
第一行输入奖章数 n (1 ≤ n ≤ 3000)
第二行输入这 n 枚奖章的卓越指数 (1 ≤ ai ≤ n)
```

### 输出描述

```
输出一个整数表示最少需要花的硬币数
```

### 测试样例

#### 样例1: 输入-输出

```
4
1 3 1 4
```

```
1
```

#### 样例2: 输入-输出

```
5
1 2 3 2 5
```

```
2
```
