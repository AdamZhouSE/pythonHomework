### 题目描述

按顺序给出n个名字，对每一个名字你需要判断该名字是否在前面出现过，若出现过，输出 YES，否则输出 NO。

### 输入描述

```
第一行输入一个数字 n，表示名字的个数。
接下来 n 行，每行输入一个非空字符串代表名字，只包含小写字母，长度不超过100。
```

### 输出描述

```
输出 n 行，若名字之前出现过，输出 YES 否则输出 NO
```

### 测试样例

#### 样例1: 输入-输出

```
6
tom
lucius
ginny
harry
ginny
harry
```

```
NO
NO
NO
NO
YES
YES
```

#### 样例2: 输入-输出

```
3
a
a
a
```

```
NO
YES
YES
```
