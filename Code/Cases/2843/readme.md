### 题目描述

b 序列有 n 个整数 b[1], b[2], ... b[n]，定义 a[i] = b[i] - b[i+1] + b[i+2] - ... b[n]。

现给定 a[1] 到 a[n] 的值，你能还原出 b 序列吗？

### 输入描述

```
第一行为序列的长度 n (2 ≤ n ≤ 100000)
第二行为 n 个数表示 a[i] (-10^9 ≤ a[i] ≤ 10^9)
```

### 输出描述

```
输出 b 序列，两两间用空格隔开
```

### 测试样例

#### 样例1: 输入-输出

```
5
6 -4 8 -2 3
```

```
2 4 6 1 3
```

#### 样例2: 输入-输出

```
5
3 -2 -1 5 6
```

```
1 -3 4 11 6
```
