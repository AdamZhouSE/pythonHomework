### 题目描述

我们把符合下列属性的数组 A 称作山脉：


	A.length >= 3
	存在 0 < i < A.length - 1 使得A[0] < A[1] < ... A[i-1] < A[i] > A[i+1] > ... > A[A.length - 1]


给定一个确定为山脉的数组，返回任何满足 A[0] < A[1] < ... A[i-1] < A[i] > A[i+1] > ... > A[A.length - 1] 的 i 的值。

### 输入描述

```
给定一个确定为山脉的数组
3 <= A.length <= 10000
0 <= A[i] <= 10^6
A 是如上定义的山脉
```
### 输出描述

```
返回任何满足 A[0] < A[1] < ... A[i-1] < A[i] > A[i+1] > ... > A[A.length - 1] 的 i 的值。
```
