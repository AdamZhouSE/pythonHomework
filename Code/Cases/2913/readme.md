### 题目描述

有一个长度为 n 的数组 a。在一个操作中，你可以选择两个元素 ai 和 aj (i≠j)，并将它们分别减少 1。 经过多次这个操作后，你可以得到全为零的数组吗？

### 输入描述

```
第一行为数组的长度 n (2≤n≤10^5)
第二行为 n 个整数表示数组元素 (1≤ai≤10^9)
```

### 输出描述

```
如果能让数组元素全为零，输出 YES，否则输出 NO
```

### 测试样例

#### 样例1: 输入-输出

```
4
1 1 2 2
```

```
YES
```

#### 样例2: 输入-输出

```
6
1 2 3 4 5 6
```

```
NO
```
