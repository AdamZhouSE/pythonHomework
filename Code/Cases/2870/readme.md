### 题目描述

现在给定 n 个整数，请使用其中任意个整数（每个只能使用一次），得到一个最大的为偶数的和。

### 输入描述

```
第一行为整数的个数 n (1 ≤ n ≤ 100000)
第二行为这 n 个整数 (1 ≤ ai ≤ 10^9)
```

### 输出描述

```
输出一个偶数表示最大的为偶数的和
```

### 测试样例

#### 样例1: 输入-输出

```
3
1 2 3
```

```
6
```

#### 样例2: 输入-输出

```
5
999999999 999999999 999999999 999999999 999999999
```

```
3999999996
```
