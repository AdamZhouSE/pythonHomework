### 题目描述

给定一个二叉树，判断其是否是一个有效的二叉搜索树。

假设一个二叉搜索树具有如下特征：


	节点的左子树只包含小于当前节点的数。
	节点的右子树只包含大于当前节点的数。
	所有左子树和右子树自身必须也是二叉搜索树。

### 输入描述

```
一个二叉树
```
### 输出描述

```
判断其是否是一个有效的二叉搜索树,是输出true,否输出false。
```

### 测试样例
#### 样例1: 输入-输出-解释
```
[5,1,4,null,null,3,6]
```
```
false
```
```
    5
   / \
  1   4
     / \
    3   6

根节点的值为 5 ，但是其右子节点值为 4 。
```
