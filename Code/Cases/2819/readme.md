### 题目描述

学校有 n 个小组外出游玩，第 i 个小组有 si (1 ≤ si ≤ 4) 个人，同学们想要打出租，每辆车最多坐4个人，同个小组必须坐在同一辆车上，求最少需要多少辆车才能让同学们都坐上出租车

### 输入描述

```
第一行为小组数 n (1 ≤ n ≤ 1e5)
第二行为 n 个数表示每个小组的人数 si (1 ≤ si ≤ 4)，
```

### 输出描述

```
输出一个整数表示至少需要的出租车数
```

### 测试样例

#### 样例1: 输入-输出

```
5
1 2 4 3 3
```

```
4
```

#### 样例2: 输入-输出

```
8
2 3 4 4 2 1 3 1
```

```
5
```

#### 样例3: 输入-输出

```
9
1 4 1 1 1 4 3 4 3
```

```
6
```
