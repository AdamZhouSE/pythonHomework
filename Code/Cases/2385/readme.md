### 题目描述
给定正整数N，计算长度为N的所有可能的不同二进制字符串，以确保没有连续的1。输出答案对（10 ^ 9 + 7）进行取模。
### 输入描述
输入的第一行包含一个整数T，表示测试用例的数量。 T测试用例的描述如下。 每个测试用例都包含一个代表二进制字符串长度的整数N。

1 ≤ T ≤ 100

1 ≤ N ≤ 100
### 输出描述
打印二进制字符串的计数数目，长度不连续1。

例如，3对应的字符串是（00,01,10），5对应的字符串是（000，001，010，100，101）
### 输入范例
```
2
3
2
```
### 输出范例
```
5
3
```
