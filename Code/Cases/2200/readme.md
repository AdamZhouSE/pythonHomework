### 题目描述

有一个只包含小写字母，长度为n的字符串S。有一些字母是好的,剩下的是坏的。

定义一个子串S_l. . .r(这里的l,r是下标)是好的，当且仅当这个子串包含不超过k个坏的字母。

求有多少个不同的满足以下要求的字符串T :

	T作为S的子串出现过。
	存在一个T出现的位置[l,r],满足S_l...r(这里的l,r是下标)是好的。

### 输入描述

```
第一行有一个字符串S。
第二行有一个字符串B。若Bi ='1'则表示Si是好的,否则表示Si是坏的。
第三行有一个整数k。
```
### 输出描述

```
一个整数：答案。
```
