### 题目描述
给定一个非负数N和两个值L和R。问题是将N的二进制表示形式的L到R范围内的位切换，即，从最右边的L位切换到最右边的R位。切换操作将位0翻转为1，将位1翻转为0。
### 输入描述
输入的第一行包含一个整数T，它表示测试用例的数量。然后是T测试用例。每个测试用例的第一行包含三个以空格分隔的整数N，L和R。
### 输出描述
对于每个测试用例，打印以N的二进制表示形式将位从最右边的Lth比特切换到最右边的Rth比特获得的数字。
### 输入范例
```
2
17 2 3
50 2 5
```
### 输出范例
```
23
44
```
