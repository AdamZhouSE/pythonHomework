
## 题目描述
给定字符串S和文本T。输出字符串S中具有文本T所有字符的最小窗口。字符串S和文本T都包含小写英文字母。
## 输入描述
输入的第一行包含一个整数T，表示测试用例的数量。然后是T测试用例。每个测试包含2行，其中包含字符串S，下一行包含文本T。1 <= T <= 100；
1 <= |N|, |X| <= 1000
## 输出描述
输出包含文本所有字符的字符串的最小窗口。如果此窗口不存在或无法完成此任务，则打印-1。
## 样例输入
2

timetopractice

toc

zoomlazapzo

oza
## 样例输出
toprac

apzo
