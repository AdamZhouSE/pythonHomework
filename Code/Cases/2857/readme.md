### 题目描述

给你一个由 n 个整数组成的数组 a。 你的任务是找到数组中所有元素的公约数的数量。 

例如，如果数组 a 为 [2,4,6,2,10]，则 1 和 2 为这个数组的公约数（因此答案为2）。

### 输入描述

```
第一行为数组的长度 n（1≤n≤4⋅10^5）
第二行为 n 个整数 (1≤ai≤10^12）
```

### 输出描述

```
输出一个整数表示数组公约数的数量
```

### 测试样例

#### 样例1: 输入-输出

```
5
1 2 3 4 5
```

```
1
```

#### 样例2: 输入-输出

```
6
6 90 12 18 30 18
```

```
4
```
