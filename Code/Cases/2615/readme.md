### 题目描述
给定的是一个长度为L的字符串。任务是从给定的字符串中找到最长的字符串，该字符串的字符按其ASCII码的降序排列并以算术级数排列。希望常见的差异应尽可能小（至少1），并且字符串的字符应具有较高的ASCII值。
### 输入描述
输入的第一行包含一个整数T，表示测试用例的数量。每个测试包含一个长度为L的字符串。
### 输出描述
对于每个测试用例，打印最长的字符串。
### 输入范例
```
2
ABCPQR
ADGJPRT
```
### 输出范例
```
RQP
JGDA
```

可能有两个最大长度的字符串-“ CBA”和“ RPQ”。但他希望该字符串具有较高的ASCII值，因此输出为“ RPQ”。
