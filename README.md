### 研究简要说明

满分率超过`97%`,分数不具有代表性

题目 `0-1分布` 通过/不通过

通过：满分

不通过：非满分，抄袭，面向用例，非python代码

根据通过率，提交次数，代码行数确定难度系数

使用随机算法和遗传算法进行智能组卷
