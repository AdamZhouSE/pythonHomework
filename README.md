### 研究简要说明

满分率超过`97%`,分数不具有代表性

题目 `0-1分布` 通过/不通过

通过：满分

不通过：非满分，抄袭，面向用例，非python代码

第$j$位同学对第$i$道题通过情况 
$$S_{ij} = \{0,1\}$$

要求做题人数$A_i$

实际做题人数$B_i$

通过率

$$P_i=\frac{\sum_{j=1}^{B_i}{S_{ij}}}{A_i}$$

总提交次数$C_i$

上传率  
$$U_i=\frac{C_i}{B_i}$$

类型z中题目通过率与上传率的相关系数
$$\rho_{z_{su}}=\frac{E\{[S-E(S)][U-E(U)]\}}{\sqrt{V(S)V(U)}}$$

不通过率作为难度系数基准
$$M_i=(1-P_i)$$
映射到区间 `[1,5]`
$$M_i'=(1+\frac{5-1}{M_{max}-M_{min}}(M-M_{min}))$$
根据上传率修正难度
$$N_i=U_i\rho_{z_{su}}$$
映射到区间 `[1,5]`
$$N_i'=(1+\frac{5-1}{N_{max}-N_{min}}(N-N_{min}))$$
难度系数
$$D_i=0.91*M_i'+0.09*N_i'$$
